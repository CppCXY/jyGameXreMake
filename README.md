#金庸群侠传XreMake

##概述

    游戏是汉家松鼠工作室的金庸群侠传X的非官方重制版
    此游戏无任何商业用途。 游戏内部许多素材均来自于互联网，若有侵犯您的权益，请与我联系。
    本游戏仅用于给爱好游戏开发和喜欢金庸类游戏的开发者做为学习参考，禁止使用本游戏中任何资源应用到任何商业行为中。

    本开源项目遵循MIT协议，也欢迎有兴趣的朋友共同改进版本。

    游戏完全使用lua开发，对lua语言的要求较高。但是提供了非常低要求的excel编辑开发模式。具体参考说明文档。
##下载

    因为游戏资源比较大，不便于传到github上，所以游戏资源单独传到百度网盘中方便下载。
    链接：https://pan.baidu.com/s/1eN3XTkyLCLVRsuv682BPJg 
    提取码：g9vx 
    下载后 解压得到res文件，分别复制到 gameDebug文件夹和gameRelease文件夹内即可。
    
##依赖

    如果游戏不能运行，请安装最新的微软常见运行库合集。
    要理解我的lua写法，必须学习我的另一个开源项目 [luaClass](https://github.com/CppCXY/luaClass)

##平台

    编译： visual studio 2019 community，android studio 3.4。

    lua开发： vscode，插件emmylua。

    lua版本：luajit 2.1

    引擎：cocos2dx3.17.2 

##开源程度

    并没有开放C++源代码，主要是C++部分就是cocos2dx引擎，我没有做更改，游戏99%采用lua开发，工具链使用python3.7开发。而lua部分和python部分都是完全开源的。
